Hello World

Dec 18, 2018

This is my personal blog and I will most of the time use this as a sort of parking space for thoughts and projects.
Some will be about workrelated stuff but more likely it will be about my shift to focus on containers and Kubernetes - plus fun activities out and about with my family.

Instead of using one of the more boring blog engines out on the internet (looking at you Blogger and Wordpress) I opted for a Markdown based, static generated blog framework hosted on github pages. I opted for Hexo.io which is very much like Jekyllrb and Hugo, though something just felt more right for me with Hexo.

Setting up github pages was a breeze and the same for setting up an apex domain to point at my Github Page, i.e. just a github repository. I especially love that in github you can set the Custom Domain option in the web UI or just include a CNAME file in your first commit containing the URL to use (in my case transforming darkcloud.github.io to darkcloud.dk)
For comments i use Gitments because it uses Github and it is free, compared to Disqus who does have a free version, but then you’ll get spam instead.
The theme is based on Louis Barranqueiro’s very excellent tranquilpeak theme. Seriously, look at his documentation. I wish everyone was as thorough.
